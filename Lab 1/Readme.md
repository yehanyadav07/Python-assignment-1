Project Name: Calorie Calculator

Description: This is a simple Python program that calculates total calories based on food items and their calorie values. It helps users track their daily calorie intake easily.

How to Run:

Make sure you have Python installed.

Open the file calorie_calculator.py in PyCharm or any code editor.

Run the file by pressing the Run button or typing this command in the terminal:

python calorie_calculator.py

Features:

Calculates total calories from user input

Allows adding multiple food items

Easy to use and modify

Example Input:

Enter food item: Apple Enter calories: 95 Add more? (y/n): y Enter food item: Rice Enter calories: 200 Add more? (y/n): n

Example Output:
